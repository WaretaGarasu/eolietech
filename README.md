# 📡 Eolietech.com — Bilingual Landing Page

This repository contains the source code for the **Eolietech** landing page, a modern and high-performance website built for an electrical firm based in the Aeolian Islands. 🏝️

The project was designed and developed with a focus on extreme speed, minimalist design, and native bilingual support.

## ✨ Key Features

- **⚡ Lightning Fast:** Built with 100% vanilla HTML, CSS, and JavaScript. No heavy frameworks or external dependencies to ensure near-instant load times.
- **🇮🇹/🇬🇧 Native Bilingual Support:** Integrated client-side localization system allowing users to switch between Italian and English seamlessly.
- **📱 Fully Responsive:** Optimized for all screen sizes, from mobile devices to high-resolution desktops.
- **♿ Accessibility Focused:** Uses semantic HTML and high-contrast typography to ensure the content is readable and accessible to everyone.
- **🔒 Privacy First:** Self-hosted assets and no third-party tracking scripts.

## 📁 Project Structure

The project maintains a simple and flat structure for easy maintenance:

- **`/` (Root):** Contains the production-ready HTML files (`index.html`, `404.html`) and site-wide configurations (`robots.txt`, `sitemap.xml`).
- **`/source/`**:
  - **`style.css`**: Custom CSS architecture with a focus on modern layouts and clean typography.
  - **`script.js`**: Lightweight JavaScript handling the bilingual logic and UI interactions.
  - **`/img/`**: Optimized icons and branded assets.
  - **`/fonts/`**: Self-hosted Inter font files to eliminate layout shifts and external calls.

## 🚀 Deployment

The site is hosted on **Cloudflare Pages**, benefiting from global edge distribution. Any push to the main branch is automatically optimized and deployed to the production environment.
